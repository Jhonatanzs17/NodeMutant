Instrucciones de ejecucion:
1-abrir una terminal en la ubicacion del proyecto
2-una vez en la ubicacion del proyecto ejecutar el comando "npm run dev"
3-utilizando cualquier herramienta como postman, consumir el api con un metodo post a la siguiente direccion: "http://localhost:3000/mutant/"
4-el metodo recibe un objeto JSON con el formato: {"dna":[strings nxn]} 
  ejm:{"dna":["ATGCGA","CAGTGC","TTATGT","AGAAGG","CCCCTA","TCACTG"]}
